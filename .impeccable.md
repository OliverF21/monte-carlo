## Design Context

### Users
Finance students using this tool for statistical analysis coursework. They need to run Monte Carlo simulations on real stock data, interpret confidence bands, and validate model accuracy through backtesting. Context is educational but the outputs inform real analytical thinking — the tool needs to feel credible, not toy-like.

### Brand Personality
**Confident. Clear. Warm.**
A modern fintech application that takes the work seriously while remaining approachable. Think Bloomberg's analytical depth married to Notion's spatial clarity and warmth. Not cold or clinical — there's room for subtle personality in transitions, states, and feedback moments.

### Aesthetic Direction
- **Visual tone**: Modern fintech — clean data visualization, strong typographic hierarchy, purposeful use of color to communicate meaning (green/red for directional, accent for interaction)
- **References**: Notion (bold simplicity, warmth in neutrals), Linear (precision, motion quality), Stripe (information density done elegantly)
- **Anti-references**: Bloomberg terminal (too dense/cold), Robinhood (too gamified), generic Bootstrap dashboards (soulless)
- **Theme**: Light mode primary with warm-tinted neutrals. Not sterile white — surfaces should feel like quality paper, not hospital walls.

### Design Principles
1. **Data is the hero** — Charts and metrics dominate. Chrome recedes. Every pixel of UI should serve comprehension.
2. **Confidence through clarity** — Students must trust the outputs. Clear labels, honest states ("could not converge" not "error"), visible methodology.
3. **Warm precision** — Professional enough for academic work, warm enough that it doesn't feel like a government form. Subtle color, generous spacing, considered typography.
4. **Motion with purpose** — Transitions communicate state changes (loading, results appearing, errors). Never decorative animation for its own sake.
5. **Progressive disclosure** — Simple on the surface, detailed underneath. Configuration is approachable; calibration details are available but not overwhelming.
